# Flask Website Template
Simple website structure/template to kickstart your Flask website

## Setup
Deploy to Heroku or run locally.

### Local Run
1. Install Dependencies `pip3 install -r requirements.txt`
2. `flask run`

### Heroku Run
1. Create Heroku
2. Create a app via CLI
3. Push to Heroku
